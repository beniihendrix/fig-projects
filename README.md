### 1. Install Git

Git is a version control system that lets you download, update, and contribute to this project. It works on **Windows, macOS, and Linux**.

#### **Windows**

1. Go to [https://git-scm.com](https://git-scm.com)
2. Click **Download for Windows**
3. Install with default options

#### **macOS**

Open the **Terminal** and run:

```bash
git --version
```

If Git is not installed, macOS will prompt you to install it.

#### **Linux (Ubuntu)**

```bash
sudo apt install git
```

After installing, confirm:

```bash
git --version
```
